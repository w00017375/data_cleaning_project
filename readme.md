# Data cleaning project

streamlit link: https://datacleaningproject-pbebitrdhqjggrngwiptxh.streamlit.app/

To run the program
```
pip install -r requirements.txt
streamlit run app.py
```

The data can be imported in the next file expansions:
- xlsx
- csv
- json
- google sheet

Key functionality
- Handle missing values by clearing and replacing them
- Duplicates removement
- Standardization and normalization of the numeric values
- Categorical column transformation including trimming, cleaning, splitting and etc.
- Creating new column by using formulas
- Data validation by custom parameters
- Visualization with optional AI suggestion
- Data export via .xlsx, .csv, .json formats as well as the information of transformation logs
- Importing transformation logs to replicate actions with a new dataframe 


The result dataframe can be exported in as:
- xlsx
- csv
- json
